## Usage

### In Chakra UI Components

```tsx
import { Box, Text } from "@chakra-ui/react"

// Using semantic color tokens
<Box bg="materialBg" borderColor="divider">
  <Text color="text.primary">Main content</Text>
  <Text color="text.secondary">Supporting information</Text>
</Box>

// Using UI colors
<Box bg="blue" color="white">
  Blue background with white text
</Box>

// Using state colors
<Button isDisabled bg="disabled.grey">
  Disabled Button
</Button>
```

### Color Accessibility

- **Text Colors**: Ensure sufficient contrast ratios between text and background colors
- **Primary Text** (#333333) on **White** (#FFFFFF) provides excellent readability
- **Secondary Text** (#748598) should be used for less important information
- **UI Colors**: Use sparingly for interactive elements and status indicators

### Semantic Usage

| Color | Usage |
|-------|-------|
| **Primary** | Main brand color, primary actions |
| **Blue** | Links, information states |
| **Green** | Success states, positive actions |
| **Yellow** | Warning states, caution |
| **Red** | Error states, destructive actions |
| **Disabled** | Inactive elements, disabled states |
