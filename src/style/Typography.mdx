# Typography System

Our typography system provides consistent text styles for both Chinese and English content.

## Font Families

### Noto Sans TC
Primary font for Chinese content with excellent CJK character support.

### Poppins
Primary font for English content, falling back to Noto Sans TC for Chinese characters.

## Usage

### In Chakra UI Components

```tsx
import { Text, Heading } from "@chakra-ui/react"

// Using Noto Sans TC styles
<Heading textStyle="notoH24">大標題 Main Heading</Heading>
<Text textStyle="notoP16">這是一段中文內容的範例文字。</Text>

// Using Poppins styles
<Heading textStyle="poppinsH20">English Heading</Heading>
<Text textStyle="poppinsP14">This is an example of English body text.</Text>

// Using decorated styles
<Text textStyle="notoP16D">帶有下劃線的重要文字</Text>
```

## Typography Guidelines

### Language-Specific Usage

1. **Chinese Content**: Use `noto*` text styles for optimal CJK character rendering
2. **English Content**: Use `poppins*` text styles for better Latin character aesthetics
3. **Mixed Content**: Choose based on the primary language of the content

### Accessibility

- Maintain a clear hierarchy with appropriate heading levels
- Ensure sufficient contrast between text and background colors
- Use larger text sizes (16px+) for body content
- Reserve 12px text for non-critical information only

### Line Height Guidelines

All text styles include appropriate line heights for optimal readability:
- Headings: 1.2x font size
- Body text: 1.5x font size
